About the System


The purpose of this system is to make the processing of the application easier and less tedious, with enhanced security and automate some common features.

The UI of the application is user-friendly and guides the user through the process.

The signatures uploaded on the server are encrypted and can only be decrypted using the same key.

The subject is automatically predicted, hence the user doesnâ€™t have to worry about it. It also has a feature to check appropriate grammar of the application.




Guidelines:


For the user:

STEP 1: Sign up using your Email ID and password.
STEP 2: Log in using the credentials.
STEP 3: Click on create a new application.
STEP 4: Enter the data for the application.
STEP 5: Click on Submit.

For the signatory:

STEP 1: Sign up using your Email ID and password.
STEP 2: Submit your Signature.
STEP 3: Click "Generate Key", a key will appear in the popup, copy it.
(NOTE:The key will be used in future to sign any application)
STEP 4: Click on Register.
STEP 5: Check pending applications by clicking the Pending Applications button.
STEP 6: Select an application for signing.
STEP 7: Type or enter the key above your name in the application.
STEP 8: Click on Submit.
